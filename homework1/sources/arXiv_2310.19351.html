Semi- and Weakly-Supervised Domain Generalization for Object Detection

Ryosuke Furuta and Yoichi Sato
The University of Tokyo
{furuta, ysato}@iis.u-tokyo.ac.jp
Abstract

Object detectors do not work well when domains largely differ between training and testing data. To solve this problem, domain generalization approaches, which require training data with ground-truth labels from multiple domains, have been proposed. However, it is time-consuming and labor-intensive to collect those data for object detection because not only class labels but also bounding boxes must be annotated. To overcome the problem of domain gap in object detection without requiring expensive annotations, we propose to consider two new problem settings: semi-supervised domain generalizable object detection (SS-DGOD) and weakly-supervised DGOD (WS-DGOD). In contrast to the conventional domain generalization for object detection that requires labeled data from multiple domains, SS-DGOD and WS-DGOD require labeled data only from one domain and unlabeled or weakly-labeled data from multiple domains for training. We show that object detectors can be effectively trained on the proposed settings with the same student-teacher learning framework, where a student network is trained with pseudo labels output from a teacher on the unlabeled or weakly-labeled data. The experimental results demonstrate that the object detectors trained on the proposed settings significantly outperform baseline detectors trained on one labeled domain data and perform comparably to or better than those trained on unsupervised domain adaptation (UDA) settings, while ours do not use target domain data for training in contrast to UDA.

1 Introduction

Object detection has been attracting much attention because it has practically useful applications such as autonomous driving. Recent object detectors have performed tremendously well on commonly used benchmark datasets for object detection, such as MSCOCOÂ [15] and PASCAL VOCÂ [5]. However, the performance of the detectors significantly drops when they are deployed on unseen domains, i.e., when the training and testing domains are different. For example, [10] reported a performance drop caused by the difference in image styles, andÂ [13] showed one caused by the weather and time difference in the images captured by car-mounted cameras.

To solve the problem, many researchers have recently been exploring unsupervised domain adaptive object detection (UDA-OD)Â [4, 13, 2]. In UDA-OD, we train object detectors using source domain data with ground-truth labels (bounding boxes and class labels) and unlabeled target domain data to adapt the detectors to the target domain. However, in the real world, target domain data cannot always be accessed in the training phase.

Domain generalizable object detection (DGOD) is another popular approach for solving the problem of the performance drop caused by the domain gapsÂ [14, 35]. In DGOD, we train object detectors using labeled data from multiple domains so that the detectors work well on unseen domains. However, it is labor-intensive to collect these data for object detection because both bounding boxes and class labels for all objects in the images must be annotated. Although single-DGODÂ [32, 6, 25, 31, 28], where we train object detectors to generalize unseen domains using labeled data from one single domain, has recently been tackled, the performance gain is still limited.

In this paper, we propose two new tasks as more realistic settings: i) semi-supervised DGOD (SS-DGOD) and ii) weakly-supervised DGOD (WS-DGOD). As shown in Fig.Â 1, the goal of SS-DGOD is to generalize object detectors to unseen domains using labeled data only from one single domain and unlabeled data from multiple domains. Note that the target domain data is not included in the training data. In WS-DGOD, we use weakly-labeled data from multiple domains instead of the unlabeled data in SS-DGOD. Here, â€œweakly-labeledâ€ means that we have only image-level labels that show the existence of each class in each training image and do not have bounding box annotations.

We show that object detectors can be effectively trained on the proposed settings with the same student-teacher learning framework, where a student network is trained with pseudo labels output from a teacher on the unlabeled or weakly-labeled data. However, if we train the detectors naively by the student-teacher learning framework, the detectors are heavily biased by the labeled domain data at the beginning of the training. To suppress the bias, we propose incorporating instance normalization (IN) layersÂ [24], which normalize the image style features in the feature maps, into the detectors.

The experimental results demonstrate that the object detectors trained on the proposed settings perform significantly better than baseline models trained on Single-DGOD and comparably to or better than ones trained on unsupervised domain adaptation settings, while ours do not access target domain data in the training phase. Also, we experimentally show that IN layers boost the performance of the detectors on the proposed settings by suppressing the bias of image styles from the labeled domain. Although Pan et al.Â [19] and Jing et al.Â [11] already proposed incorporating IN layers for domain generalization, we found that it does not always improve the performance on Signle-DGOD and DGOD in our experiments. In contrast, our experiments show that IN layers constantly improve the performance on the proposed SS-DGOD and WS-DGOD settings.

Our contributions are summarized as follows:

  • â€¢

    We propose two new problem settings: SS-DGOD and WS-DGOD.

  • â€¢

    We show that object detectors can be effectively trained on the proposed settings with the same student-teacher learning framework.

  • â€¢

    We propose incorporating IN layers to improve the performance of the detectors trained on the proposed settings by suppressing the bias to the labeled domain in the student-teacher learning framework.

  • â€¢

    Our experiments show that the detectors trained on the proposed settings perform comparably to or better than those trained on the related settings.

3 Related Works

3.1 Domain Generalization for Object Detection

Many methods have been proposed for domain generalization for image classification as summarized in recent survey papersÂ [37, 27]. However, domain generalization for object detection has not been explored widely, compared with image classification.

Lin et al.Â [14] proposed an approach to disentangle domain-specific and domain-invariant features by adversarial learning on both image-level and instance-level features for DGOD. Liu et al.Â [16] tackled DGOD in underwater object detection and proposed DG-YOLO. For Single-DGOD, Wang et al.Â [31] proposed a self-training method that utilizes the temporal consistency of objects in videos. Wu et al.Â [32] proposed a method that disentangles domain-invariant features by contrastive learning and self-distillation. Fan et al.Â [6] proposed perturbing the channel statistics of feature maps, which can be interpreted as data augmentation of image styles to a variety of domains. Wang et al.Â [28] proposed a disentangle method on frequency space for object detection from unmanned aerial vehicles. Recently, Vidit et al.Â [25] proposed an augmentation approach using a pre-trained vision-language model (CLIP) with textual prompts.

Unlike all the above methods, as discussed in Sects.Â 1 andÂ 2, we propose and tackle new problem settings called SS-DGOD (Semi-Supervised Domain Generalization for Object Detection) and WS-DGOD (Weakly-Supervised Domain Generalization for Object Detection), which are more practical than DGOD. Furthermore, our experiments show that the object detectors trained on SS-DGOD and WD-DGOD outperform the state-of-the-art method for Single-DGODÂ [25] by utilizing the unlabeled (or weakly-labeled) data.

3.2 UDA for Object Detection

As summarized inÂ [18], various methods have been proposed for UDA-OD. There are two main streams for UDA-OD, i.e., alignment-basedÂ [3, 21] and student-teacher learning-basedÂ [4, 13], but the recent trend is the latter approach. In this paper, we show that the proposed two new settings (SS-DGOD and WS-DGOD) can be solved by the student-teacher learning framework. Also, we show that the object detectors trained on the proposed settings can perform comparably to or better than those trained on the UDA-OD setting.

Another approach to make detectors robust to unseen domains is an online adaptation using test imagesÂ [26]. However, this approach requires extra computation for online updates at test time in contrast to our settings.

3.3 Semi-Supervised Domain Generalization

There are a few existing methods that utilize both labeled and unlabeled data for domain generalization (SSDG) on image classificationÂ [36, 38]. Zhang et al.Â [36] proposed an unsupervised pretraining method called DARLING, which performs contrastive learning on unlabeled images to obtain domain-irrelevant feature representation. Zhou et al.Â [38] extended a semi-supervised learning method called FixMatchÂ [23] to SSDG.

To the best of our knowledge, this paper is the first attempt to tackle SSDG for object detection. In addition, we tackle the â€œweakly-labeledâ€ setting (i.e., WS-DGOD), which is not defined on image classification.

3.4 Student-Teacher Learning

Several existing works have studied the use of the student-teacher learning frameworks for a variety of tasks such as domain generalization on image classificationÂ [33], (in-domain) weakly-supervised object detectionÂ [30], (in-domain) semi-supervised object detectionÂ [17], (in-domain) semi-supervised image classificationÂ [23], and UDA for semantic segmentaionÂ [1, 29, 9, 34]. Note that our aim is not to propose a technically novel method. Instead, we show that the proposed new settings can be effectively solved by adopting the student-teacher learning framework.

4.3.4 Normalize Image Style Features

By the student-teacher learning framework described in Sects.Â 4.1-4.3, we can effectively train object detectors on both SS-DGOD and WS-DGOD. 5.2 Implementation Details

We used soft pseudo labeling proposed inÂ [2] for student-teacher learning. As the object detector, we used Gaussian FasterRCNNÂ [2], in which the regression output is modified to adopt the soft labels. As the backbone network, we used ResNet101Â [8] and incorporated IN layers at the ends of three residual blocks, following IBNNet_bÂ [19]. We applied the same hyperparameters toÂ [2] except for the number of iterations. All training (including baseline models) was done with four Tesla V100 GPUs. The parameters of the backbone network were initialized with the ResNet101 pre-trained on ImageNet except for the IN layers. The IN layers, RPN, and RoIhead were initialized with random parameters. During the inference (testing) phase, we used the teacher network.

On the car-mounted camera dataset, the detectors were trained with 20,000 and 40,000 iterations for the pretraining and the student-teacher learning of SS-DGOD (or WS-DGOD), respectively. During the training, we saved the models and evaluated the performance on the validation at every 4,000 iterations, and the best model was used for the evaluation. The whole training took about two days. For fair comparisons, the compared models on Single-DGOD and DGOD were trained with 60,000 iterations, and the best models at the validation of every 4,000 iterations were used for evaluation.

On the artistic style image dataset, we performed the same procedure for training, validation, and evaluation, but the numbers of iterations for the pretraining and the student-teacher learning were set to 10,000 and 20,000 respectively, and the validation was conducted at every 2,000 iterations. The whole training took about one day. For fair comparisons, the compared models on Single-DGOD and DGOD were trained with 30,000 iterations, and the best models at the validation of every 2,000 iterations were used for evaluation.

6 Conclusions

In this paper, we proposed two new problem settings called semi-supervised domain generalizable object detection (SS-DGOD) and weakly-supervised DGOD (WS-DGOD) to train object detectors that can generalize to unseen domains. Compared with the existing DGOD settings, their advantage is that the labeled data from multiple domains are not required for training because the detectors are trained with the labeled data from one domain and unlabeled (or weakly-labeled) data from multiple domains. In contrast to unsupervised domain adaptive object detection (UDA-OD) settings, they do not need the target domain data during the training. Therefore, they are more practical than the existing problem settings.

We showed that the object detectors can be effectively trained on the proposed two settings with the same student-teacher learning framework. Also, we proposed incorporating instance normalization (IN) layers into the detection networks to suppress the bias to the labeled domain data. The experimental results showed that the detectors trained on SS-DGOD and WS-DGOD performed significantly better than the baseline models. It was also confirmed that IN layers improved the performance of the detectors on the two proposed settings. In addition, they performed comparably to or better than the models trained on UDA-OD setting while they did not use the target domain data during the training.

In future work, we are planning to investigate the performance of recent Transformer-based detectors on SS-DGOD and WD-DGOD settings.
